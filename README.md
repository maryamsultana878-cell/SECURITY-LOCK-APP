# Smart Mobile Security System Using Face Recognition and Intruder Detection

## Overview

Smart Mobile Security System is a security-focused application designed to protect user accounts and devices through multiple layers of authentication and monitoring. The system combines traditional login methods with face recognition, OTP verification, activity tracking, and security management features.

The application helps users detect unauthorized access attempts, manage trusted faces, receive security alerts, and maintain detailed activity records.

## Objectives

* Provide secure user authentication and account management.
* Detect unauthorized access attempts using face recognition.
* Send instant alerts for suspicious activities.
* Maintain activity and login history.
* Support password recovery through OTP verification.
* Enhance overall device and account security.
